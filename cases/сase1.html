Вариационный анализ

Вариационный анализ чувствительности параметра LADD к вариабельности входящих в его расчет компонентов

by Konstantin Osipov (26.04 - 30.04 2021)

При решении задач связанных с оценкой рисков здоровью населения из-за воздействия промышленных предприятий используется параметр LADD – среднесуточная доза поступления вещества в организм человека в течение его жизни. Формула для расчета имеет следующий вид: \[LADD=\frac{C_{эл}{\cdot}ED{\cdot}EF{\cdot}V}{BW{\cdot}AT{\cdot}365}\] причем параметры, входящие в расчет имеют вероятностный характер распределения в соответствии со следующией таблицей:

Параметр Характеристика Тип распределения Вариабельность (доверительный интервал значений)
V скорость дыхания внутри помещений, \(м^3/час\) логнормальное 70 (65-105)
EF частота воздействия, дней/год триангулярное 350 (180-365)
ED продолжительность воздействия, лет логнормальное 30 (19-43)
BW масса тела, кг логнормальное 70 (55-80)
AT период осреднения экспозиции, лет константа 70
\(C_{эл}\) концентрация элемента, \(мг/м^3\) нормальное \(C_{эл}\) ± SD

Поставленная задача состояла в том, чтобы задавая по очереди каждый коэффициент в виде соответствующего вероятностного распределения построить вероятностное распределение результата - самого параметра \(LADD\). (Метод Монте-Карло). Тоесть каждый параметр по очереди задается в виде массива значений (n=2000) с указанной таблицей функцией распределения, а остальные параметры остаются константами.

Part2

Создадим таблицу (data.frame) с 5-ю колонками, в каждой из которых находится результат расчета \(LADD\) при вариации одного из коэффициентов, а остальные заданы константами из таблицы.

for (i in c(1:5)) {tmp <- const
  tmp[i] <- var_s[i]
  LA <- tmp$C_Cr*tmp$V*tmp$ED*tmp$EF/(tmp$BW*tmp$AT*365) #Расчет LA
  if (i == 1) {res_df <- data.frame(LA)} #Создаем data.frame из одной колонки
  else { res_df <- cbind(res_df, LA)} #И заполняем его
}
colnames(res_df) <- paste0("var=", names(var_s))

Преобразуем в формат Long для дальнейшего построения графиков:

long_res <- res_df %>% 
  pivot_longer(cols = everything(), names_to = "Variate", values_to = "LA") %>% 
  arrange(Variate)
long_res$Variate <- as.factor(long_res$Variate)

И строим зависимости полученных значений LA в виде плотности распределения вероятности на одном графике для сравнения.

ggplot(data = long_res, aes(x=LA)) + geom_density(aes(fill = Variate), alpha=0.4) +
  annotate(geom = "text", x=100, y=0.125, 
           label = paste("LA==", expression(frac(C[Cr]%.%V%.%ED%.%EF,BW%.%AT%.%365))), parse=TRUE) +
  annotate(geom = "table", x = -20, y = 0.23, label = list(anno_df)) + 
  scale_x_continuous(limits = c(-25, 175), breaks = seq(-25, 175, 25)) +
  xlab(label = expression(paste(LA %*% 10^-6)))

Обратите внимание, что при моделировании была допущена ошибка: когда параметр EF задавался константой, его значение было равно 350. А когда он задавался в виде нормального распределения, то его среднее было несколько меньше и равно 270. Так и из графика хорошо видно что при вариации EF результат расчета LA cмещен в меньшую сторону, как раз на величину пропорциональную разности этих значение.

Для полной наглядности построим этот график в виде Выборочной функции распределения (CDF). Что дает нам легко получить вероятность достижения LADD того или иного значения.

Заключение

Из рассмотрения этих графиков мы можем сделать вывод, что наибольшее влияние на расчетный параметр LA имеет коэффициент \(С_{эл}\) как собственно и должно быть, чему также способствует правильно выбранная размерность и масштаб остальных коэффициентов.
